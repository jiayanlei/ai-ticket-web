# AI 智能工单分析指挥中心核心指标布局优化

## 变更摘要

- 将首页大屏左侧“核心指标”卡片从上下结构调整为左右结构。
- 左侧展示指标标题与趋势信息，右侧固定宽度展示核心数值与单位。
- 放大核心数字字号，增强运营监控大屏的数据视觉权重。
- 数字区域设置固定宽度、右对齐和垂直居中，减少右侧空白并提升信息密度。
- 针对嵌入态、全屏态和低高度场景补充响应式尺寸，避免文字裁切和数字压缩。
