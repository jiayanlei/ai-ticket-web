# Visual QA Checklist

Use a 1920x1080 browser viewport.

- [ ] Header appears at 72px and all controls fit.
- [ ] Sidebar appears at 280px and menu labels do not truncate awkwardly.
- [ ] Main dashboard has no blank chart panels.
- [ ] Every route renders a high-fidelity enterprise page.
- [ ] Glassmorphism panels use consistent dark background, border, and blur.
- [ ] Tables, cards, buttons, tags, and charts do not overlap.
- [ ] Text remains readable and does not overflow buttons or panels.
- [ ] Active route, hover state, warning state, success state, and danger state are visually distinct.
- [ ] AI Copilot, risk, SLA, and alert surfaces are visible in relevant pages.
- [ ] Basic narrower desktop viewport remains usable.
