# AI Ticket OS Master Design Prompt

Design a world-class enterprise SaaS admin system named **AI Ticket OS**.

Product tagline: **Enterprise AI Customer Service Operating System**.

The product serves large enterprise customer service centers, call centers, after-sales teams, and customer success organizations. It is an AI-native operation platform that merges patterns from OpenAI, Linear, Jira, Salesforce, Zendesk, ServiceNow, HubSpot, Datadog, and Notion.

## Design Language

- Future Enterprise SaaS
- AI Native First
- Glassmorphism
- Cyber Tech
- Dark Mode
- Data Intelligence
- Operation Command Center

## Canvas

- Primary desktop canvas: 1920x1080.
- Header height: 72px.
- Sidebar width: 280px.
- Desktop-first responsive behavior.
