# Chart And Component Prompts

## Chart Prompt

```text
Design an enterprise data visualization component for AI Ticket OS.
Use dark glass panels, electric blue/cyan/purple accents, precise labels, live status markers, and dense but readable operational data.
The chart must be directly implementable with ECharts.
```
