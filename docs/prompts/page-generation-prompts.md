# Page Generation Prompts

Use this pattern for each AI Ticket OS page:

```text
Create a high-fidelity dark enterprise SaaS page for AI Ticket OS.
Page: [PAGE NAME]
Audience: [PRIMARY ROLE]
Goal: [BUSINESS OUTCOME]
Include: KPI cards, smart filters, advanced table, command panel, AI insight card, real-time chart, status/risk indicators, and page-specific operational modules.
Style: Future Enterprise SaaS, AI Native First, Glassmorphism, Cyber Tech, Dark Mode, Data Intelligence.
Canvas: 1920x1080, header 72px, sidebar 280px.
Use English UI copy.
```

## Core Page Emphasis

- Dashboard: service volume, channel state, AI health, SLA, alerts, sentiment, geography, and trend intelligence.
- Ticket Center: lifecycle, escalation, approval, archive, merge, split, AI recommendations, SLA and relationship graph.
- Call Center: queue, IVR, agent wall, monitor, whisper, barge-in, recording, QA, transcript, sentiment and risk.
- AI Agent Center: marketplace, orchestration, monitoring, logs, performance, cost, tools, trace, workflows, and multi-agent graph.
- Data Cockpit: CEO, operations, support, and AI operations cockpit modes.
