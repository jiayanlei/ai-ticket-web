# Prototypes

Static high-fidelity prototype images for AI Ticket OS live here.

## Naming

Use this naming pattern for 1920x1080 screens:

- `01-dashboard.png`
- `02-ticket-center.png`
- `03-call-center.png`
- ...
- `30-open-platform.png`

The `source/` directory stores editable source files or prompt outputs used to generate prototype images.
