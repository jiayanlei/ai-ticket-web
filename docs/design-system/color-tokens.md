# Color Tokens

| Token | Value | Usage |
| --- | --- | --- |
| Primary | `#4F7BFF` | Primary actions, active route, key lines |
| Secondary | `#8B5CF6` | AI, models, orchestration |
| Accent | `#00E5FF` | Live signal, realtime highlights |
| Success | `#22C55E` | Healthy, resolved, online |
| Warning | `#F59E0B` | SLA risk, pending, warning |
| Danger | `#EF4444` | Incidents, breach, critical |
| Background | `#070B14` | App background |
| Card | `#0F172A` | Solid panels |
| Glass | `rgba(255,255,255,0.06)` | Glass panels |
| Border | `rgba(255,255,255,0.12)` | Subtle panel borders |
