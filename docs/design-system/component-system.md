# Component System

## Required Components

- Data Card: dense business facts with owner, status, and trend.
- Metric Card: KPI value, unit, trend, and status tone.
- KPI Card: executive-level metric with target comparison.
- Glass Card: reusable panel with dark glass background and border.
- Agent Card: human or AI agent identity, status, skill, load, quality score.
- Customer Card: customer value, risk, lifecycle, sentiment, and latest touchpoint.
- Alert Card: severity, source, SLA, escalation, owner, and action.
- Timeline: lifecycle, audit, conversation, and workflow history.
- Activity Feed: real-time events and operational changes.
- Command Panel: actions, bulk operations, and AI-generated recommendations.
- AI Chat Panel: assistant suggestions, draft replies, citations, and tool calls.
- Advanced Table: filters, tags, status, risk, SLA, and row actions.
- Smart Filter: presets, date range, channel, team, owner, status, priority, SLA.
- Drawer: detail, edit, approval, and audit side surfaces.
- Side Sheet: persistent right context panel.
- Workflow Builder: nodes, branches, tools, model calls, approval flow.
- Knowledge Editor: article/FAQ editor with AI improvement and review.
- Agent Monitor: live runs, health, latency, cost, and trace.
- Real-time Chart: live operational trend with stable sizing.
