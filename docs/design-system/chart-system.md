# Chart System

Charts are implemented with ECharts and share dark command-center styling.

## Required Types

- Line chart
- Area chart
- Bar chart
- Stacked bar chart
- Donut chart
- Radar chart
- Funnel chart
- Sankey chart
- Relationship graph
- Customer journey map
- Organization chart
- Knowledge graph
- Agent call chain graph
- Real-time monitor chart
- Heatmap
- Map analysis

## Rules

- Every chart must have visible labels or legends.
- Empty chart panels are not acceptable in high-fidelity pages.
- Use blue, cyan, purple, green, amber, and red for meaning, not decoration alone.
