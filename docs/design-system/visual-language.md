# Visual Language

AI Ticket OS uses a dark enterprise command-center interface.

## Principles

- Dense but readable information.
- Operational first, decorative second.
- Glass panels with real hierarchy.
- AI-native surfaces visible across pages.
- Cyber-tech accents reserved for live status, risk, and AI activity.
- No marketing landing-page composition inside the admin product.

## Layout

- Header: 72px.
- Sidebar: 280px.
- Main surface: full viewport operational workspace.
- Cards: radius 8px or less.
- Tables: compact, scannable, status-rich.
- Charts: stable dimensions and non-empty states.
