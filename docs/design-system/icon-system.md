# Icon System

Use Ant Design Vue icons for implementation.

## Icon Categories

- Navigation: dashboard, service, channel, operations, AI, customer, knowledge, analytics, system.
- Action: search, filter, add, edit, archive, merge, split, approve, export, refresh.
- State: online, idle, busy, warning, critical, healthy, breached.
- AI: robot, model, workflow, prompt, trace, cost, tool call.

Icons must be paired with tooltips for icon-only buttons.
