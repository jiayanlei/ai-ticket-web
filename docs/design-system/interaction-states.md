# Interaction States

## Entity States

- Ticket: New, Triage, Assigned, In Progress, Pending Customer, Escalated, Resolved, Archived.
- SLA: Healthy, Watch, At Risk, Breached.
- Agent: Online, Busy, Idle, Away, Offline.
- AI: Running, Stable, Degraded, Failed, Optimizing.
- Alert: Info, Warning, High, Critical, Resolved.

## Interaction Rules

- Buttons must show clear disabled/loading/hover/focus states.
- Dangerous actions require confirmation.
- AI-generated content must show confidence or source context when relevant.
- Real-time panels should show live status and last updated time.
