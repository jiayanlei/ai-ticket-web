# AI Ticket OS

Enterprise AI Customer Service Operating System.

AI Ticket OS is a Vue 3 + Vite + TypeScript enterprise SaaS admin system for large support centers, call centers, after-sales operations, and customer success teams.

## Product Direction

- Future Enterprise SaaS
- AI Native First
- Glassmorphism
- Cyber Tech
- Dark Mode
- Data Intelligence
- Operation Command Center

## Tech Stack

- Vue 3
- Vite
- TypeScript
- Ant Design Vue
- ECharts
- Pinia
- Vue Router
- SCSS

## Key Experience

- 72px global command header.
- 280px enterprise sidebar.
- 30 high-fidelity AI Ticket OS pages.
- Dynamic menu and permission registration.
- Dark enterprise visual system with glass panels.
- Reusable route-driven page engine for metrics, filters, charts, work queues, AI recommendations, workflow panels, and realtime events.
